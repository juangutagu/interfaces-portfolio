Bienvenidos a esta página web

En esta página web encontrarás todas las tareas que hemos hecho durante el curso de Desarrollo de Interfaces Web.

Es una web responsiva que se adapta a los dispositivos móviles.


En el lateral izquierdo puedes customizar el color de fondo de la sección principal y el texto de la página web, cambiando su tamaño, color y fuente.

También puedes elegir entre temas preconfigurados con diferentes colores; O activar el modo "Color simple", que visualizará la página web completa en blanco y negro.
